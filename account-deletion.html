Account Deletion – Shoqëria e Fjalëve

Users of the Shoqëria e Fjalëve mobile application can request the deletion of their account and associated data at any time.

How to request account deletion

  1. Send an email to fjaleshloje@gmail.com
  2. Use the subject line: Account Deletion Request
  3. Include the email address used to create your account

Data that will be deleted

  • User account information (Firebase Authentication)
  • Game progress and profile data associated with the account

Data retention period

All data will be permanently deleted within 30 days after the request is verified.

If any data must be retained for legal or security reasons, it will be stored securely and deleted as soon as permitted.
